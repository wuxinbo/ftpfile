android studio ftpfile
本项目是使用android studio完成的，这个app 主要实现根据用户提供的用户名和密码来访问远程的FTP服务器（暂时支持端口号21的服务器），
登陆成功后，用户便可以从服务器上下载文件（目前支持单个文件的下载，后续的功能留待以后），也可以读取内存设备中的文件上传到服务器
中（此功能还在开发中）。
